# Image Manipulator

Image Manipulator is a tool to add basic filters to Image. 

### Website:
[Image Manipulator](https://jashnimje.github.io/Image_Manipulator/)

## Features

### Filters

* Grayscale.
* Sepia.
* Rainbow.
* Blur.
* Colorize (Any Colors).

### Green Screen Editor (Beta).
